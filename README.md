2.사진 리소스가 세로가 아닌 가로로 추가되어 문제가 있었음
-WinForms은 휴대폰 사진의 Exif 회전 정보를 무시함. 즉, 세로 사진이 WinForms에선 가로로 인식함
사진을 메모장으로 다시 저장해서 해결
